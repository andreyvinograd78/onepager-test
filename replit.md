# Andrey Vinograd - Personal One-Pager Website

## Overview
A minimal, premium one-page credibility hub for Andrey Vinograd, co-founder at Gogrow. The site is designed to make it easy for founders/operators (subscription apps & games) and investors/capital partners to understand who Andrey is and connect quickly.

## Project Structure
```
├── public/
│   ├── index.html      # Main HTML page
│   ├── styles.css      # Styling (light premium theme)
│   └── script.js       # Smooth scroll animations
├── server.js           # Express server (port 5000)
├── package.json        # Node.js dependencies
└── spec.md             # Original design specification
```

## Tech Stack
- Node.js with Express for serving static files
- Vanilla HTML, CSS, JavaScript (no framework)
- Google Fonts (Inter)

## Running Locally
The website runs on port 5000 via the "Website" workflow:
```bash
node server.js
```

## Design Features
- Light premium color palette (warm off-white background)
- Refined typography with Inter font
- Subtle fade-in animations
- Mobile-responsive layout
- "Connect on LinkedIn" as primary CTA
